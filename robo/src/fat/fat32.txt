MBR - 512 bytes
    Boot code - 446 bytes
    Partition {1..4} - 16 bytes each
        Boot flag - 1 byte
        CHS begin - 3 bytes
        Type code (0x0B or 0x0C for FAT32) - 1 byte
        CHS end - 3 bytes
        LBA begin (partition start) - 4 bytes
        # of sectors - 4 bytes
    Sanity - 2 bytes
        0x55
        0xAA

FAT32 Volume ID (located at LBA) - 512 bytes
    0x0B (Bytes per sector, always 512) - 2 bytes
    0x0D (Sectors per cluster, 2^{0..7}) - 1 byte
    0x0E (# of reserved sectors, usually 0x20) - 2 bytes
    0x10 (# of FATs, always 2) - 1 byte
    0x24 (Sectors per FAT, depends) - 4 bytes
    0x2C (Root directory first cluster, usually 0x2) - 4 bytes
    0x1FE (Signature, always 0x55AA) - 2 bytes

FAT32 Partition
    Volume ID - 1 sector
    Reserved sectors - variable, usually 0x20 sectors
    FAT 1/2 - variable
    Clusters - variable

Directory entries - 32 bytes each
    Short filename
        Name - 11 bytes
        Attrib - 1 byte
        First cluster high - 2 bytes
        First cluster low - 2 bytes
        File size - 4 bytes
    Long filename
    Unused
        First byte 0xE5
    End of directory 
        First byte 0x00
